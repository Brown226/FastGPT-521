```typescript
// packages/service/common/system/constants.ts
export const EditionConst = {
  IS_EE: process.env.NODE_ENV === 'production',
  // 或通过 License 验证
};
```

### 3.2 商业功能配置项

从 `projects/app/src/service/common/system/index.ts` 中发现以下配置：

| 配置项                        | 用途           | 商业版专用 |
| -------------------------- | ------------ | ----- |
| `hideChatCopyrightSetting` | 隐藏版权信息       | ✅     |
| `isPlus`                   | 是否显示 Plus 功能 | ✅     |
| `feConfigs.lafEnv`         | Laf 环境配置     | ✅     |

### 3.3 API 路由分析

项目使用 `/proApi/` 前缀区分商业版 API：

```typescript
// projects/app/src/web/support/user/api.ts
export const oauthLogin = (params) =>
  POST('/proApi/support/user/account/login/oauth', params);
export const ssoLogin = (params) =>
  GET('/proApi/support/user/account/sso', params);
```
